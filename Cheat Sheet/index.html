HTML Table and Button Reference Sheet

Table Tags

Tag Name Description
<table> Table The <Table> element creates a table.
<thead> Table Head A table’s head is created with the <thead> element.
<tbody> Table Body A table’s body is created with the <tbody> element.
<tr> Table Row The <tr> element adds rows to a table.
<td> Table Data To add data to a row, you can use the <td> element.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
<colspan> Column Span Defines how many columns a td element should span.
<rowspan> Row Span Defines how many rows a td element should span.

Form Tags

Tag Name Description
<form> Form The purpose of a <form> is to allow users to input information and send it.
<input> Input The purpose of a <input> is to allow users to input information and send it.
<select> Select A <select> element is populated with <option> elements and renders a dropdown list selection.
<datalist> Datalist A <datalist> element is populated with <option> elements and works with an <input> to search through choices.
<textarea> Textarea A <textarea> element is a text input field that has a customizable area.

*When a <form> is submitted, the name of the fields that accept input and the value of those fields are sent as name=value pairs.

Form Attributes

Attribute Name Description
<action> Action The <form>‘s action attribute determines where the form’s information goes.
<method> Method The <form>‘s method attribute determines how the information is sent and processed.
<type> Type To add fields for users to input information we use the <input> element and set the type attribute to a field of our choosing

Type Attributes

Attribute Name Description
text Text Setting type to "text" creates a single row field for text input.
password Password Setting type to "password" creates a single row field that censors text input.
number Number Setting type to "number" creates a single row field for number input.
checkbox Checkbox Setting type to "checkbox" creates a single checkbox which can be paired with other checkboxes.
radio Radio Setting type to "radio" creates a radio button that can be paired with other radio buttons.
list List Setting type to "list" will pair the <input> with a <datalist> element if the id of both are the same.
submit Submit Setting type to "submit" creates a submit button.
